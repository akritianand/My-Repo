Online Failure Management System

Introduction

Failure management system is a set of processes used in an organization to resolve failures and provide information thereof. As manual failure management system used in the CSS now has its disadvantages in terms of efficacy, efficieny and transparency, there is a need for an online failure management system for the S&T department of Delhi Metro, which is known for its efficiency, timeliness and punctuality. This Online Failure Management System (OFMS) meets these needs and offers the following features:

Features

  • It provides a 24 x 7 available platform for failure management:
  • It makes complaints easier and more efficient to file, assign, monitor, track and resolve;
  • It lends the officials an effective and efficient tool to identify and monitor performance of complaints handling;
  • It brings in greater transparency and responsiveness to the system;
  • It controls redundancy in storing the same data multiple times; and
  • It provides for the evolution of the system according to changing needs.
